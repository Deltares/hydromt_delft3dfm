API reference#

DFlowFM model class#

Initialize#

DFlowFMModel(root, mode, config_fn, ...[, ...])

API for Delft3D-FM models in HydroMT.

Setup components#

DFlowFMModel.setup_config(**cfdict)

Update config with a dictionary.

DFlowFMModel.setup_channels(region, channels_fn)

This component prepares the 1D channels and adds to branches 1D network.

DFlowFMModel.setup_rivers_from_dem(region, ...)

This component sets the all river parameters from hydrograph and dem maps.

DFlowFMModel.setup_rivers(region, rivers_fn)

Prepares the 1D rivers and adds to 1D branches.

DFlowFMModel.setup_pipes(region, pipes_fn[, ...])

Prepares the 1D pipes and adds to 1D branches.

DFlowFMModel.setup_manholes([manholes_fn, ...])

Prepares the 1D manholes to pipes or tunnels.

DFlowFMModel.setup_1dboundary([...])

Prepares the 1D boundary_type boundaries to branches using timeseries or a constant for a specific branch_type at the boundary_locs locations.

DFlowFMModel.setup_bridges([bridges_fn, ...])

Prepares bridges, including bridge locations and bridge crossections.

DFlowFMModel.setup_culverts([culverts_fn, ...])

Prepares culverts, including locations and crossections.

DFlowFMModel.setup_mesh2d(region[, res])

Create an 2D unstructured mesh or reads an existing 2D mesh according UGRID conventions.

DFlowFMModel.setup_mesh2d_refine([...])

Refine the 2d mesh within the geometry based on polygon polygon_fn or raster samples sample_fn.

DFlowFMModel.setup_link1d2d([...])

Generate 1d2d links that link mesh1d and mesh2d according UGRID conventions.

DFlowFMModel.setup_maps_from_rasterdataset(...)

This component adds data variable(s) from raster_fn to maps object.

DFlowFMModel.setup_maps_from_raster_reclass(...)

This component adds data variable(s) to maps object by combining values in raster_mapping_fn to spatial layer raster_fn.

DFlowFMModel.setup_2dboundary([...])

Prepares the 2D boundaries from line geometries.

DFlowFMModel.setup_rainfall_from_constant(...)

Prepares constant daily rainfall_rate timeseries for the 2D grid based on constant_value.

DFlowFMModel.setup_rainfall_from_uniform_timeseries(...)

Prepares spatially uniform rainfall forcings to 2d grid using timeseries from meteo_timeseries_fn.
