Attributes#

DFlowFMModel.region

Returns geometry of region of the model area of interest.

DFlowFMModel.crs

Returns coordinate reference system embedded in region.

DFlowFMModel.res

Resolution of the mesh2d.

DFlowFMModel.root

Path to model folder.

DFlowFMModel.config

Model configuration.

DFlowFMModel.staticmaps

Model static maps.

Model forcing.

Model states.

Model results.
